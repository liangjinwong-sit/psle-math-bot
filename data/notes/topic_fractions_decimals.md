# Topic: Fractions and Decimals

## Key Ideas
- A fraction shows equal parts of a whole.
- A decimal is another way to represent part of a whole.
- Fractions and decimals can be compared, added, subtracted, and converted.
- Equivalent fractions have the same value.
- To compare fractions, use a common denominator or convert to decimals when suitable.
- To compare decimals, line up the digits by place value.
- Always check whether the question is asking for comparison, operation, or conversion.

## Standard Methods

### Method 1: Convert a fraction to a decimal
1. Divide the numerator by the denominator.
2. Write the decimal carefully.

Example: 3/4 = 3 ÷ 4 = 0.75

### Method 2: Convert a decimal to a fraction
1. Write the decimal as a fraction over 10, 100, or 1000.
2. Simplify the fraction if possible.

Example: 0.6 = 6/10 = 3/5

### Method 3: Add or subtract fractions with the same denominator
1. Keep the denominator the same.
2. Add or subtract the numerators.
3. Simplify if needed.

Example: 2/7 + 3/7 = 5/7

### Method 4: Add or subtract decimals
1. Line up the decimal points.
2. Add or subtract carefully.
3. Write the decimal point in the answer.

Example: 3.45 + 1.2 = 4.65

### Method 5: Compare fractions or decimals
1. Use equivalent fractions, a common denominator, or decimals.
2. Compare the values carefully.
3. Use the correct symbol: >, <, or =.

## Question Patterns
- If the question asks to change a fraction into a decimal, divide the numerator by the denominator.
- If the question asks to change a decimal into a fraction, write it over 10, 100, or 1000 and simplify.
- If fractions have the same denominator, add or subtract the numerators only.
- If decimals are being added or subtracted, line up the decimal points.
- If the question asks which is greater or smaller, compare place values or use equivalent fractions.
- If the question gives a word problem with parts of a whole, think about whether a fraction or decimal model is more useful.

## Alternative Phrasings
- equal parts of a whole = fraction
- decimal number = decimal
- same value = equivalent
- change into = convert
- bigger than = greater than
- smaller than = less than
- line up by place value = align decimal points

## Common Pitfalls
- Adding denominators when fractions already have the same denominator.
- Forgetting to line up decimal points.
- Comparing decimals by the number of digits instead of place value.
- Not simplifying a fraction when needed.
- Mixing up numerator and denominator.
- Writing a decimal-to-fraction conversion incorrectly.

## Worked Example 1
Question: Convert 3/4 to a decimal.

Step 1: Divide 3 by 4.
Step 2: 3 ÷ 4 = 0.75.

Answer: 0.75

Why this works: A fraction can be written as a division statement.

## Worked Example 2
Question: Convert 0.8 to a fraction.

Step 1: Write 0.8 as 8/10.
Step 2: Simplify 8/10 by dividing top and bottom by 2.
Step 3: 8/10 = 4/5.

Answer: 4/5

Why this works: A decimal in the tenths place can be written over 10 and simplified.

## Worked Example 3
Question: Find 2/9 + 4/9.

Step 1: The denominators are the same.
Step 2: Add the numerators: 2 + 4 = 6.
Step 3: Keep the denominator 9.

Answer: 6/9 or 2/3

Why this works: Fractions with the same denominator can be combined by adding the numerators.

## Worked Example 4
Question: Find 5.6 - 2.3.

Step 1: Line up the decimal points.
Step 2: Subtract 2.3 from 5.6.
Step 3: 5.6 - 2.3 = 3.3.

Answer: 3.3

Why this works: Decimal subtraction follows place value, so the decimal points must line up.

## Worked Example 5
Question: Which is greater, 0.45 or 0.5?

Step 1: Write 0.5 as 0.50.
Step 2: Compare 0.45 and 0.50 by place value.
Step 3: 0.50 is greater than 0.45.

Answer: 0.5 is greater.

Why this works: Writing both decimals to the same number of places makes comparison easier.
